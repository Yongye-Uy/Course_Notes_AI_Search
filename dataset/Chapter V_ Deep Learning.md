3. How Does Deep Learning Work?
● A neural network is a computational model that consists of multiple layers of
interconnected nodes or neurons
● Each neuron takes input from the previous layer, applies a nonlinear activation
function, and produces an output that is passed on to the next layer
● During training, the network adjusts its weights and biases to minimize the
error between its predictions and the true labels
● Deep learning models typically require large amounts of labeled data and
computationally intensive training, but can generalize well to new data

5. Types of deep learning techniques
There are several types of deep learning techniques, including:
● Convolutional Neural Networks (CNNs): These are commonly used for image and
video recognition tasks. They learn to extract features from raw pixel data by
applying filters or kernels to the input image and then pooling the results.
● Recurrent Neural Networks (RNNs): These are used for tasks that involve sequence
data, such as speech recognition and language translation. They use feedback loops
to maintain a memory of past inputs and produce outputs that depend on the
current input as well as the previous inputs.
● Deep Belief Networks (DBNs): These are a type of generative model that can learn
to represent complex data distributions in an unsupervised way. They are used in
applications such as image and audio recognition, and natural language processing.
8

5. Types of deep learning techniques
● Autoencoders: These are neural networks that are trained to reconstruct their
input data from a compressed representation. They can be used for tasks
such as dimensionality reduction, feature extraction, and anomaly detection.
● Multi-layer Perceptrons (MLPs): These are the basic building blocks of deep
neural networks. They consist of multiple layers of interconnected neurons
and can be used for a wide range of tasks, including classification, regression,
and clustering.
● Generative Adversarial Networks (GANs): These are a type of neural network
that can generate new data samples that are similar to a training dataset.
They consist of two networks: a generator network that creates new samples,
and a discriminator network that distinguishes between real and fake
samples.
